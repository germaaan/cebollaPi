¿Por dónde empiezo?

Por instalar un sistema operativo en la Raspberry Pi

Sistemas operativos disponibles:

  • Raspbian
  • Ubuntu MATE
  • Snappy Ubuntu
  • Pidora
  • SARPi
  • Arch Linux ARM
  • Gentoo Linux
  • FreeBSD
  • Kali Linux

Instalar Raspbian

Modo salvaje: usando dd
                      
dd bs=4M if=raspbian.img of=/dev/[sdX|mmcblkX]

Instalar Raspbian

Modo SinQuebraderosDeCabeza: usando NOOBS (New Out Of the Box Software)
  • Formateamos la tarjeta SD en FAT32
  • Descargamos NOOBS: https://downloads.raspberrypi.org/NOOBS_latest
  • Descomprimimos el archivo
  • Copiamos los archivos a la tarjeta SD
  • Encendemos la Raspberry Pi e instalamos mediante el instalador

Todos sabemos que mola más instalarlo a mano, pero falla más que una escopeta de feria.

¿Qué tenemos que configurar?

  • Acceso SSH a la Raspberry Pi
  • Interfaces de red
  • Servidor DHCP
  • Punto de acceso
  • iptables
  • Tor

Acceso SSH a la Raspberry Pi

                  
sudo raspi-config
              
              
"Advanced options" ➡️ "SSH" ➡️ "Enable or disable ssh server" ➡️ YES

Actualizar sistema e instalar paquetes necesarios

                    
sudo apt update && sudo sudo apt full-upgrade -y

sudo apt install hostapd isc-dhcp-server tor
