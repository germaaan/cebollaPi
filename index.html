¿Qué es Tor?

Tor es una red que nos permite acceder a Internet de forma anónima y también el software que nos permite utilizar dicha red.

Oculta el origen y el destino del tráfico de Internet, así nadie sabe fácilmente quién eres y qué estás viendo.

Uso práctico: saltarse la censura de algunos páises.

Servidor DHCP

                    
Configuramos el servidor DHCP para que asigne IP a las máquinas que se conectan a la Raspberry Pi.

                    
/etc/dhcp/dhcpd.conf

# Cambiar el nombre del servidor y la dirección de los DNSs
option domain-name "local";
option domain-name-servers 8.8.8.8, 8.8.4.4;

# Descomentar esta opción para indicar que va a ser el único
# servidor DHCP
authoritative;

# Añadir configuración para la red
subnet 192.168.1.0 netmask 255.255.255.0 {
  range 192.168.1.160 192.168.1.170;
  option subnet-mask 255.255.255.0;
  option broadcast-address 192.168.1.255;
  option routers 192.168.1.150;
}



                

                

                    
Interfaces de red

                    
La interfaz a través de la que nos vamos a conectar en la Raspberry Pi tiene que tener una dirección IP estática para hacer de servidor DHCP.

                    
/etc/network/interfaces

allow-hotplug wlan0

#iface wlan0 inet manual
#wpa-roam /etc/wpa_supplicant/wpa_supplicant.conf

iface wlan0 inet static
  address 192.168.1.150
  netmask 255.255.255.0



                

                

                    
Interfaces de red

                    
La interfaz a través de la que nos vamos a conectar en la Raspberry Pi es la conexión WiFi.

                    
/etc/default/isc-dhcp-server
            
INTERFACES="wlan0"



                

                

                    
Punto de acceso

                    
Nuestra Raspberry Pi será visible como una conexión de red inalámbrica.

                    
/etc/hostapd/hostapd.conf
  
# Interfaz a través de la que nos vamos a conectar
interface=wlan0

# Driver de la tarjeta de red
driver=[nl80211|rtl871xdrv|...]

# Nombre de la subred
ssid=HACKERMAN

# Usar la banda de frecuencia a usar (2.4GHz)
hw_mode=g

# Canal a usar
channel=6

# Acepta todas las direcciones MAC
macaddr_acl=0

# Usar autenficación WPA
auth_algs=1

# Solo se pueden conectar los que conozcan el nombre de la red
ignore_broadcast_ssid=0

# Usa WPA2
wpa=2

# Usa una clave precompartida
wpa_key_mgmt=WPA-PSK

# Contraseña de la red
wpa_passphrase=clavesupersegura

# Usar cifrado AES
rsn_pairwise=CCMP



                

                

                    
iptables

                    
Activar reenvío de paquetes IP

                    
/etc/sysctl.conf
  
net.ipv4.ip_forward=1

sudo sysctl -p


                

                

                    
iptables

                    
Configurar tablas de reglas

                    
              
sudo iptables -F
sudo iptables -t nat -F
sudo iptables -t nat -A PREROUTING -i wlan0 -p tcp --dport 22 -j REDIRECT --to-ports 22
sudo iptables -t nat -A PREROUTING -i wlan0 -p udp --dport 53 -j REDIRECT --to-ports 53
sudo iptables -t nat -A PREROUTING -i wlan0 -p tcp --syn -j REDIRECT --to-ports 9040
sudo sh -c 'iptables-save > /etc/iptables.ipv4.nat'



                

                

                    
iptables

                    
Usar nuevas reglas

                    
/etc/network/interfaces
              
up iptables-restore /etc/iptables.ipv4.nat



                

                

                    
Tor

                    
Configurar el enrutamiento a través de la red Tor

                    
/etc/tor/torrc
              
Log notice file /var/log/tor/notices.log
VirtualAddrNetwork 10.192.0.0/10
AutomapHostsSuffixes .onion,.exit
AutomapHostsOnResolve 1
TransPort 9040
TransListenAddress 192.168.1.150
DNSPort 53
DNSListenAddress 192.168.1.150



                

                

                    
Tor

                    
Crear archivos de registro

                    

sudo touch /var/log/tor/notices.log
sudo chown debian-tor /var/log/tor/notices.log
sudo chmod 644 /var/log/tor/notices.log



                

                

                    
Iniciar todos los servicios

                    
                    
# Para esta sesión
sudo service isc-dhcp-server start
sudo service hostapd start
sudo service tor start

# Automáticamente con cada inicio del sistema
sudo update-rc.d isc-dhcp-server enable
sudo update-rc.d hostapd enable
sudo update-rc.d tor enable
